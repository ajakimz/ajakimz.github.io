Welcome

This site represents my Project 2 work for Universal Design for Digital Media. It explores the foundational ideas behind Universal Design and how these principles influence digital accessibility, interaction design, and inclusive online experiences.

Project Overview

This project breaks down Universal Design into three interconnected areas: what it is, why it matters, and how designers can apply it. The purpose of this website is to present those ideas in a clear, accessible, and thoughtfully structured way (following the same principles the project explains).

Instead of simply summarizing course modules, this project reinterprets them into a multi-page site that practices semantic HTML, readable typography, consistent layout, and a user-first approach. Each page focuses on one guiding question and demonstrates how Universal Design influences modern digital design.

What You’ll Find Here

The site is divided into three pages:

  • What: Defines Universal Design and explains how it differs from accessibility.
  • Why: Examines the ethical, cultural, and practical reasons inclusive design is essential.
  • How: Provides actionable methods, design strategies, and implementation examples.

Project Intent

My goal is to create a resource that is informative, approachable, and aligned with inclusive practices. The visual structure, content organization, and writing tone are all crafted to support clarity and usability for a wide range of readers.

Feel free to navigate using the menu and explore each area of Universal Design through structured, accessible content.
