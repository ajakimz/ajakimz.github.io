Guess the Album
Guess the album and artist based on the cover below!

Extension Explanation:
For my quote machine extension, I created a “Guess the Album” generator using the iTunes Search API. Each click shows a random album cover, and users have to guess the artist or album name before revealing the answer. It combines music data with a fun guessing-game format.

API in My Critique Project:
In my critique project, I could imagine using an API to bring in interactive or visual data. For example, pulling accessibility stats or user interface data from a public API could help visualize how inclusive certain design systems are.
